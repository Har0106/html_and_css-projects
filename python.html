Python

Python is a high-level, interpreted, general-purpose programming language. It supports multiple programming paradigms, including structured, object-oriented and functional programming. Python is dynamically-typed and garbage-collected. It was designed by Guido van Rossum. It was first released on 20th February 1991 as Python 0.9.0. Python 2.0 was released in 2000 and Python 3.0 was released in 2008.
